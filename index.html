HTML Tag: <a>

The <a> tag defines a hyperlink, which is used to link from one page to another.

Attributes:

  • href: Specifies the URL of the page the link goes to.
  • target: Specifies where to open the linked document.

Example:

<a href="https://example.com" target="_blank">Visit Example</a>

Usage Notes:

Use the target="_blank" attribute to open the link in a new tab or window.

HTML Tag: <div>

The <div> tag defines a division or section in an HTML document.

Attributes:

  • id: Specifies a unique id for the <div>.
  • class: Specifies one or more class names for the <div>.

Example:

<div id="main-content" class="container">Content here</div>

Usage Notes:

The <div> tag is often used as a container for other elements.

CSS Property: color

The color property sets the color of text.

Values:

  • color-name: E.g., red, blue.
  • hex-code: E.g., #ff0000.
  • rgb: E.g., rgb(255,0,0).

Inheritable:

Yes, the color property is inheritable.

Default Value:

Depends on the user agent, but typically black (#000000).

Applicable To:

All elements.

Example:

p { color: blue; }

Usage Notes:

Use this property to set the color of your text.

CSS Property: background-color

The background-color property sets the background color of an element.

Values:

  • color-name: E.g., red, blue.
  • hex-code: E.g., #ff0000.
  • rgb: E.g., rgb(255,0,0).
  • rgba: E.g., rgba(255,0,0,0.5) (with transparency).

Inheritable:

No, the background-color property is not inheritable.

Default Value:

Transparent.

Applicable To:

All elements.

Example:

div { background-color: #ff0000; }

Usage Notes:

Use this property to set the background color of your elements.

CSS Property: margin

The margin property sets the outer space around an element.

Values:

  • length: E.g., 10px, 2em.
  • percentage: E.g., 10%.
  • auto: Automatically calculates the margin.
  • inherit: Inherits the margin value from the parent element.

Inheritable:

No, the margin property is not inheritable.

Default Value:

0 for all sides.

Applicable To:

All block-level elements and inline-block elements.

Example:

div { margin: 10px; }

Usage Notes:

Use this property to create space around elements, outside of any defined borders.
